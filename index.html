Welcome to our Plegma API Pages

Welcome to the Yodiwo Platform! We are an IoT Platform-as-a-Service (PaaS) provider with a unique connectivity framework, based on the first IoT Editor that uses graphs to codelessly interconnect models of Devices and existing Apps with each other.

API walkthrough

You can find a fairly comprehensive walk-through (among other things) of our API and how to use it to communicate with our cloud servers here.

API references

You can also inspect the formal reference for:

- Plegma: our southbound API

- Warlock: our northbound API

- User Code API: available from within Cyan's C# blocks

- mNode Plugin API: used to create you own custom plugins for WisperPro's mNode agent
