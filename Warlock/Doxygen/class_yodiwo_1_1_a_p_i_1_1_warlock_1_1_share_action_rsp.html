Detailed Description

Share Action Response class. Each ShareThingsRsp message contains the same number of these as in the ShareThingsReq message it is responding to, and each ShareActionRsp.Id corresponds to a ShareActionReq.Id

Member Data Documentation

int Yodiwo.API.Warlock.ShareActionRsp.Id

Id of request that this response refers to

ShareInfo [] Yodiwo.API.Warlock.ShareActionRsp.Info

(refers to eShareActionType.Ask requests only) Array of ShareInfo that referenced ThingKey is currently shared with. May be empty.

Is left empty for eShareActionType.Share and eShareActionType.Unshare requests where the request outcome is derived from the GenericRsp.IsSuccess and GenericRsp.Message fields

bool Yodiwo.API.Warlock.ShareActionRsp.IsPending

If true, the respective request was not completed because of an action pending on the receiver's part. In such a case PendingToken will be set to a value that will be provided as reference in a future ShareNotifyMsg
