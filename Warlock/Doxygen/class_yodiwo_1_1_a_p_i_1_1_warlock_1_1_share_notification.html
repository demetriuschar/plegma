Detailed Description

Notification message; may inform about:

  • something that happened successfully and Warlock subscribers need to be informed (PendingToken is empty and IsSuccessful is true
  • a share event that is pending and requires user action (PendingToken contains a token that can be used in a subsequent "answer" Share Request)
